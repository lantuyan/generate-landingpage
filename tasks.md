## Instructions for AI Agent

Để hoàn thành mỗi task:

1. Đọc file `prompt.md` trong thư mục tương ứng
2. Tạo các file sau trong cùng thư mục:
   - `index.html` - HTML structure
   - `styles.css` - CSS styling
   - `script.js` - JavaScript functionality
3. Đảm bảo landing page:
   - Là ONE-PAGE design (single scrolling page)
   - Tuân theo design style được chỉ định
   - Responsive trên các thiết bị
   - Có animations/interactions phù hợp với style
4. Sau khi hoàn thành, cập nhật status trong file này thành `done`
